# carrotandstick
## 메이커스4기 당근과 채찍 - 목표 달성 어플 구글 플레이스토어 출시 

<p align="left">
목표를 효과적으로 달성할 수 있도록 도와줍니다

포기의 연속인 당신의 목표.
귀여운 당근과 함께 매일 매일 달성 해보세요.
하루 물 2L 마시기, 유산소 30분 등과 같이 매일 이루고자 하는 목표를 설정하고
매일 체크해주세요.
매일 체크할 때마다 미지에 쌓인 당근 컬렉션이 지급됩니다.
매일 노력하는 나에게 줄 선물을 입력해서, 동기부여를 얻으세요!
</p>
